# Projeto — Banco de Dados SQL (DDL + DML)
Este projeto foi desenvolvido para a disciplina de Banco de Dados do Senac. O objetivo foi criar um banco fictício utilizando comandos DDL (definição do banco) e DML (manipulação dos dados).

## Objetivo

- Criar tabelas usando comandos DDL.
- Manipular dados usando comandos DML.
- Compreender o funcionamento de criação, inserção, atualização e exclusão.

## Comércio escolhido

Cafeteria CoffeeDream, com tabelas de produtos, clientes e pedidos.

## Como executar
- Baixe ou clone o repositório.
- Abra seu SGBD (PostgreSQL, MySQL etc.)
- Execute o script dentro da pasta `/sql_scripts`.

---

## DDL — Data Definition Language

Comandos que criam ou modificam a estrutura do banco.

Exemplos:

- `CREATE TABLE`
- `ALTER TABLE`
- `DROP TABLE`

## DML — Data Manipulation Language

Comandos que manipulam os dados dentro das tabelas.

Exemplos:

- `INSERT`
- `UPDATE`
- `DELETE`
- `SELECT`
